# 안드로이드 알약 식별 챗봇 시연 시나리오

## 시연 전 준비

- 테스트 계정과 비밀번호를 미리 준비합니다.
- 식별 결과를 확인한 알약 사진을 준비합니다.
- Android 기기와 서버의 네트워크 연결 상태를 확인합니다.
- API 및 인증 서버 주소가 올바른지 확인합니다.

## 시연 시나리오 (약 3분)

### 1. 앱 실행 및 로그인

1. Android 기기에서 **알약 식별 챗봇** 앱을 실행합니다.
2. 홈 화면의 로그인 버튼을 선택합니다.
3. 준비한 테스트 계정으로 로그인합니다.

발표 멘트:

> 사용자는 계정으로 로그인한 후 알약 식별 채팅을 시작할 수 있습니다.

### 2. 알약 사진으로 검색

1. 채팅 화면에서 사진 첨부 또는 카메라 버튼을 선택합니다.
2. 카메라 화면에서 알약을 중앙에 놓고 초점을 맞춥니다.
3. 알약 표면이 너무 밝거나 어두우면 화면의 노출 조절 기능으로 밝기를 낮추거나 높입니다. 각인 문자가 선명하게 보이는 수준으로 맞춥니다.
4. 사진을 촬영한 뒤 확인 화면에서 사용할 사진을 확정합니다. 필요하면 준비한 알약 사진을 대신 선택합니다.
5. 사진을 전송하고 분석이 완료될 때까지 기다립니다.
6. 세로로 표시되는 후보 알약 카드와 이름, 제조사, 점수를 보여줍니다.
7. 필요하면 메뉴의 **이미지 분석 정보** 옵션을 변경하여 모양, 색상, 각인 정보의 표시 여부를 보여줍니다.

발표 멘트:

> 카메라의 노출을 조절하면 빛 반사나 어두운 환경에서도 알약의 색상과 각인을 더 선명하게 촬영할 수 있습니다. 사진을 전송하면 이미지 분석 결과와 유사한 후보 알약을 확인할 수 있으며, 상세 분석 정보는 앱 전체 설정으로 표시하거나 숨길 수 있습니다.

### 3. 특징 설명으로 검색

1. **새 대화** 버튼을 선택합니다.
2. 입력창에 다음 예시를 입력하거나 마이크로 말합니다.

> 초록색 arf가 새겨진 알약입니다.

3. 응답 상태가 **추가 정보 필요**로 변경되면 화면에 제공된 선택지 중 알맞은 답을 선택합니다.

발표 멘트:

> 사진이 없어도 알약의 색상, 모양, 각인 문자를 설명하여 검색할 수 있습니다. 추가 정보가 필요한 경우 챗봇이 단계별로 질문합니다.

### 4. 검색 결과 확인

1. 상태가 **결과 확인 필요**로 변경되는 것을 보여줍니다.
2. 결과가 맞으면 **맞아요**, 다르면 **아니에요**를 선택합니다.
3. 선택한 답변에 따라 다음 응답으로 진행되는 것을 보여줍니다.

발표 멘트:

> 사용자는 검색 결과를 직접 확인하고 맞는지 여부를 전달할 수 있습니다. 결과가 다르면 정답 정보를 추가로 입력할 수 있습니다.

### 5. 새 대화로 초기화

1. **추가 정보 필요**, **결과 확인 필요** 또는 **정답 ID 입력** 상태에서 **새 대화** 버튼을 선택합니다.
2. 이전 메시지와 선택한 이미지가 제거되는 것을 확인합니다.
3. 상태가 **대기 중**으로 돌아오는 것을 보여줍니다.

발표 멘트:

> 새 대화를 선택하면 진행 중이던 검색 상태와 입력 내용이 모두 초기화되어 새로운 알약 검색을 시작할 수 있습니다.

## 마무리

> 이 앱은 알약 사진과 사용자 설명을 이용해 후보를 검색하고, 필요한 정보를 대화형으로 보완할 수 있도록 구성했습니다.

## 시연 시 주의사항

- 서버 응답 시간을 고려하여 각 전송 후 잠시 기다립니다.
- 카메라 촬영 시 알약 전체가 화면에 들어오도록 하고 초점을 확인합니다.
- 알약 표면에 빛이 반사되면 노출을 낮추고, 각인이 어둡게 보이면 노출을 높입니다. 색상이 실제와 달라질 정도로 과도하게 조절하지 않습니다.
- 네트워크 문제가 발생할 경우 준비한 사진과 서버 주소를 다시 확인합니다.
- 시연 전에 동일한 사진과 설명으로 전체 흐름을 한 번 점검합니다.
