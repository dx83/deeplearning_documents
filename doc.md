# 알약 모양 분류에 EfficientNet-B0를 사용한 이유
- EfficientNet-B0는 제한된 데이터와 연산 자원 환경에서 이미지 분류 성능과 효율의 균형이 좋은 모델이다.
- 알약 이미지는 색상, 윤곽, 각인, 원형/타원형/장방형과 같은 형태 차이가 분류에 중요한 요소이다.
- EfficientNet-B0는 이미지의 시각적 특징을 단계적으로 추출할 수 있어 알약 모양 분류에 적합하다.
- ImageNet 사전학습 가중치를 활용하면 비교적 적은 알약 이미지 데이터로도 안정적인 전이학습이 가능하다.
- EfficientNet 계열 중 가장 가벼운 기본 모델이기 때문에 학습 및 추론 비용이 낮다.
- 대형 모델에 비해 과적합 부담이 상대적으로 작아 제한된 데이터 환경에 적합하다.

# PPT용 설명 문장
- 본 프로젝트에서는 알약 이미지의 형태적 특징을 효율적으로 학습하기 위해 EfficientNet-B0를 사용하였다.
- EfficientNet-B0는 EfficientNet 계열의 기본 모델로, 비교적 적은 파라미터와 연산량으로 이미지 분류에 필요한 특징을 효과적으로 추출할 수 있다.
- 특히 ImageNet 사전학습 가중치를 활용한 전이학습에 적합하여, 데이터 규모가 제한적인 알약 이미지 분류 환경에서도 안정적인 학습을 기대할 수 있다.
- 또한 알약의 윤곽, 색상, 각인 등 시각적 특징을 학습할 수 있다는 장점이 있다.

# 발표 방어용 답변
- EfficientNet-B0는 EfficientNet 계열의 기준 모델이다.
- 본 프로젝트에서는 확장 모델을 사용한 것이 아니라, 경량성과 전이학습 효율이 좋은 B0를 선택했다.
- B1 이상 모델보다 가볍기 때문에 학습과 추론이 빠르고, 제한된 환경에서도 적용하기 쉽다.
- 알약 이미지 데이터가 충분히 크지 않은 상황에서는 과도하게 큰 모델보다 B0처럼 가벼운 모델이 더 현실적인 선택이다.

# 슬라이드 핵심 Bullet
- EfficientNet 계열의 기본 모델로 구조가 검증되어 있음
- B1 이상 모델보다 가볍고 학습/추론 비용이 낮음
- ImageNet 사전학습 가중치를 활용한 전이학습에 적합
- 제한된 데이터 환경에서 과도하게 큰 모델보다 과적합 부담이 작음
- 알약의 윤곽, 색상, 각인 등 시각 특징 추출에 유리함

# EfficientNet 기본 설명
- EfficientNet은 이미지 분류 성능과 연산 효율을 동시에 고려하여 설계된 모델 계열이다.
- 기존 이미지 분류 모델보다 적은 파라미터와 연산량으로 높은 성능을 달성하는 것을 목표로 한다.
- EfficientNet-B0는 EfficientNet 계열의 기본 모델이며, 이후 B1~B7 모델은 B0를 기준으로 확장된 구조이다.
- 전이학습에 많이 사용되며, 이미지 분류 문제에서 효율적인 선택지로 활용된다.

# 발표용 EfficientNet 설명
- EfficientNet은 성능만 높이는 것이 아니라, 모델 크기와 연산량까지 고려해서 효율적으로 설계된 이미지 분류 모델이다.
- 기본 모델인 B0를 기준으로 B1부터 B7까지의 모델이 만들어졌다.
- 본 프로젝트에서는 그중 가장 가벼운 B0를 사용했다.
