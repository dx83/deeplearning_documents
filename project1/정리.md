3. 전체 시스템 구조
    - 서비스 과정
        1) 알약 이미지 업로드
        2) 외양 분류 모델: --       
           
           색상 분류 모델: 마지막 레이어를 11개의 색상분류로 변경 한 Resnet18기반 전이학습 모델
           
           각인 인식 모델 
             1. 각인 영역탐지: 알약 각인 영역 탐지에 적합하도록 파인튜닝 된 YOLO11기반 모델
             2. 각인 해석 OCR: Synth90k 데이터 셋에 트레이닝 된 CRNN 모델
             각인 영역탐지 모델을 통해 OCR를 수행 할 영역을 찾은 후 각인해석

        3) 3개의 모델을 합산해서 알약 예측
           - 모든 csv 열에 대한 외양, 색상, 문자/마크 컬럼 에 대해 다음을 계산.
           - 점수 계산법
             - (외양 신뢰도  * 외양 가중치) +  (색상 신뢰도 * 색상 가중치) + (문자/마크 신뢰도 * 문자/마크 가중치)
             - 각 외양, 색상, 문자/마크 컬럼 값과 각 모델이 예상한 값이 일치하지 않으면 그 분야의 점수는 0으로 처리
           - 점수 기준으로 모든 열을 정렬 후 가장 높은 값 3후보를 출력
        4) 챗봇으
